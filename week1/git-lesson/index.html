Git Notes

  • git init initializes an empty repository
  • git add . or name-of-fil to stage stages a file for commit
  • git commit -m "Commit message" commits changes to memory
  • git status shows status of repository
  • git log shows the commit history
  • git remote -v checks if there is a remote connection
  • git remote add origin (paste in HTTPS from "new repository" on gitHub) creates a new connection between gitHub and local repository
  • git push origin master pushes from computer repository to gitHub
  • git clone [paste in SSH link from code button in github repository] copies down a repository from gitHub
  • git diffshows changes made since laast commit ('q' to exit)
  • git branch developcreates a new branch (does not open the branch) and calls it "develop"
  • git checkout -b developdoes the create & open branch all at once
  • git push origin developpushes the commit to the branch
  • git checkout branchopens the branch
  • git checkout mainopens the main code again
